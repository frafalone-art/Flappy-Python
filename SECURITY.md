# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in FastFixIt, please report it privately.

Contact:

[assistenza@fastfixit.it](mailto:assistenza@fastfixit.it)

Please include:

* Description of the issue
* Steps to reproduce
* Potential impact
* Screenshots or proof of concept if available

## Responsible Disclosure

Please:

* Do not access data that does not belong to you
* Do not disrupt the service
* Do not publicly disclose vulnerabilities before a fix is available

We appreciate responsible security research and will investigate all legitimate reports.

## Supported Versions

FastFixIt security updates are applied to the production platform currently in service.
